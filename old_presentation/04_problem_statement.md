# Problem Statement

---

## The Core Problem

> **Children with special needs and their families lack a unified platform that addresses the complete therapeutic journey - from scheduling sessions to tracking family-wide impact, preparing children for challenging situations, and making progress tangible and rewarding.**

---

### Gap 2: Child-Only Focus

| What's Tracked | What's Ignored |
|----------------|----------------|
| Child's session attendance | Parent stress levels |
| Child's therapy goals | Sibling feelings |
| Child's clinical progress | Family dynamics |
| | Home practice challenges |
| | Household impact |

> **Research shows:** When parents are stressed, therapy outcomes for children decrease by up to 40%.

## Problem Summary

| Problem Area | Current State | Impact |
|--------------|---------------|--------|
| **Scheduling** | Phone calls, manual registers | Delays, double bookings, no real-time availability |
| **Records** | Paper-based, scattered | Data loss risk, hard to search, no insights |
| **Family Focus** | Child-only tracking | Misses stress factors, incomplete picture |
| **Preparation** | Generic advice | Anxiety, meltdowns, failed outings |
| **Motivation** | Abstract reports | Disengaged children, dropped therapy |
| **Communication** | Fragmented updates | Parents feel uninformed |
